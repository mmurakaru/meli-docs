<Info>
  **Prerequisite**: Please install Node.js (version 19 or higher) before proceeding.
</Info>

Follow these steps to install and use meli.js in your app:

**Step 1**: Install Meli:

<CodeGroup>

  ```bash npm
  npm i meli.js
  ```

  ```bash yarn
  yarn add meli.js
  ```

  ```bash pnpm
  pnpm add meli.js
  ```

</CodeGroup>

**Step 2**: Initiate meli:

```js
meli.init("MELI_TOKEN")
```
